# 代码引用复制插件

VSCode 插件，用于快速复制文件引用、行号引用和行范围引用。

## 功能

- **@文件名** - 复制整个文件引用
- **@文件名#行号** - 复制特定行引用
- **@文件名#起始行-结束行** - 复制行范围引用

## 使用方法

1. 在编辑器中右键打开上下文菜单
2. 选择对应的复制命令：
   - `复制文件引用` - 复制当前文件名
   - `复制行引用` - 复制当前光标所在行或选中区域的起始行
   - `复制范围引用` - 复制选中的行范围

## 开发

```bash
# 安装依赖
npm install

# 编译
npm run compile

# 监听模式
npm run watch
```

## 调试

1. 按 `F5` 启动调试
2. 在新窗口中测试插件功能

## 打包

```bash
# 安装打包工具
npm install -g @vscode/vsce

# 打包
vsce package
```
